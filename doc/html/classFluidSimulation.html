◆ SetBoundaryConditions()

void FluidSimulation::SetBoundaryConditions ( FluidSimulation::BoundaryType  bound,
Matrix &  medium 
) const
overridevirtual

The fluid is assumed to be contained in a box so this function assures that:

  • if a medium is a velocity: A No-Slip condition is met, and the appropriate velocity component is 0 relative to a boundary. That essentially means setting last cell near boundary minus penultimate cell (Prandtl no-slip boundary condition) (https://en.wikipedia.org/wiki/Boundary_layer https://en.wikipedia.org/wiki/No-slip_condition)
  • if a medium is a density: As stated in: https://www.dgp.toronto.edu/public_user/stam/reality/Research/pdf/GDC03.pdf we assume continuity - in case of density, last cells should be the same as penultimate cell
    Parameters
    boundwhich type of boundary contidion will be handled
    mediummedium on which boundary contidion will be set
